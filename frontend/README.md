# 家計簿アプリ - フロントエンド

React + TypeScript + Vite で構築された家計簿アプリケーションのフロントエンドです。

## 技術スタック

- **フレームワーク**: React 18
- **言語**: TypeScript
- **ビルドツール**: Vite
- **ルーティング**: React Router v6
- **HTTPクライアント**: Axios
- **スタイリング**: TailwindCSS

## 主な機能

- ユーザー認証（ログイン・新規登録）
- ダッシュボード（収支の概要表示）
- 取引管理（収入・支出の登録・一覧・削除）
- カテゴリ管理（カテゴリの作成・削除・色設定）
- 予算管理（月ごとの予算設定と進捗表示）

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 環境変数の設定

必要に応じてAPIのURLを変更してください。

```
VITE_API_BASE_URL=http://localhost:8000
```

### 3. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:5173 にアクセスしてください。

## ビルド

本番用にビルドする場合：

```bash
npm run build
```

ビルドされたファイルは `dist/` ディレクトリに出力されます。

## プロジェクト構造

```
src/
├── components/       # 共通コンポーネント
│   ├── Layout.tsx   # レイアウトコンポーネント
│   └── PrivateRoute.tsx # 認証が必要なルートの保護

│   └── AuthContext.tsx # 認証コンテキスト
├── pages/          # ページコンポーネント
│   ├── Home.tsx    # ホームページ
│   ├── Login.tsx   # ログインページ
│   ├── Register.tsx # 新規登録ページ
│   ├── Dashboard.tsx # ダッシュボード
│   ├── Transactions.tsx # 取引管理
│   ├── Categories.tsx # カテゴリ管理
│   └── Budgets.tsx # 予算管理
├── services/       # APIクライアント
│   └── api.ts      # API通信処理
├── types/          # TypeScript型定義
│   └── index.ts    # 型定義
└── App.tsx         # ルーティング設定
```

## 使い方

### 1. アカウント作成

1. トップページの「今すぐ始める」または「新規登録」をクリック
2. 名前、メールアドレス、パスワードを入力して登録

### 2. カテゴリの作成

1. ナビゲーションの「カテゴリ」をクリック
2. 「新規カテゴリ」ボタンをクリック
3. カテゴリ名、種別（収入/支出）、カラーを設定

### 3. 取引の登録

1. ナビゲーションの「取引」をクリック
2. 「新規取引」ボタンをクリック
3. 種別、カテゴリ、金額、日付、メモを入力

### 4. 予算の設定

1. ナビゲーションの「予算」をクリック
2. 「新規予算」ボタンをクリック
3. カテゴリ、予算額、対象月を設定

### 5. ダッシュボードで確認

ダッシュボードで月ごとの収入・支出・収支を確認できます。
